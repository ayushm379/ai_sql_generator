# AI SQL Generator

### Functional Requirements
- Converts English command to SQL based on the dialect.
- A reviewer agent verifies the generated SQL query.
- Security checks would be there to block sensitive information.
- Query optimization to create index, rewrite joins.
- Query cost optimizer to enforce Limit.
- Run the query and return the result.
- User authentication to secure data based on role.

### Tech Used:
- Python
- FastAPI
- Ollama
